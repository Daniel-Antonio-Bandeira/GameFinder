# 🎮 GameFinder

Aplicação web que permite buscar jogos utilizando a API da RAWG, exibindo informações como nome, nota e imagem de cada jogo.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* API REST (RAWG)

---

## 🔍 Funcionalidades

* Busca de jogos por nome
* Consumo de API externa
* Renderização dinâmica de resultados
* Exibição de:

  * 🎮 Nome do jogo
  * ⭐ Nota
  * 🖼️ Imagem


---

## 🧠 Aprendizados

Neste projeto, pratiquei:

* Consumo de APIs com `fetch`
* Uso de `async/await`
* Manipulação do DOM
* Criação dinâmica de elementos HTML
* Organização de código em funções
